# Interactive Brand Experience Game – Master Prompt Framework

## CORE INSIGHT: The Magic Formula
**What makes the Domino's experience work:**
- **Progressive challenge escalation** (simple tap → complex draw interactions)
- **Immediate sensory feedback** (visual FX, animations, sounds)
- **Score-based narrative** (customer reaction tier = emotional payoff)
- **Frictionless sharing** (unique codes incentivize viral spread)
- **Mobile-first physics** (touch-optimized, 2-5 min play time)

---

## MASTER PROMPT FOR SIMILAR EXPERIENCES

### **High-Level Brief**
```
Create a [DURATION: 2-5 minute] interactive brand game experience that:
1. Converts passive users into active brand ambassadors
2. Generates shareable moments with unique unlock codes
3. Teaches product value through gameplay (not advertising)
4. Works seamlessly on mobile (touch + mouse support)
5. Creates emotional peaks at strategic intervals
6. Incentivizes immediate action (coupon/discount claim)
```

### **Architecture Template (10-Scene Flow)**

**Phase 1: Onboarding (Scene 1)**
- Portal/magical entry (builds anticipation)
- Brand hero animation (pizza icon → pulse effect)
- Single CTA ("TAP TO ENTER")
- Duration: 3-5 seconds

**Phase 2: Skill Challenges (Scenes 2-8) – Mix & Match**
Choose 6-7 challenges from this palette:
- **Drag Interactions** (sauce spreading, dragging items)
- **Drawing/Swiping** (pizza slicing, path-tracing)
- **Physics Simulation** (dough tossing, ingredient balancing)
- **Timing Challenges** (oven heat management, speed runs)
- **Choice-Based** (topping selection, customization)
- **Tap/Reflex** (rapid tapping, pattern matching)
- **Gesture Recognition** (shake, rotate, swipe patterns)

**Phase 3: Emotional Payoff (Scenes 9-10)**
- Score-based outcome reveal (3-tier: Good/Amazing/Legendary)
- Contextual NPC reaction (emoji faces + dynamic text)
- Unique code generation (format: `[BRAND]-[RANDOM]-[TIMESTAMP]`)
- Social share prompt (WhatsApp, Twitter, Instagram, SMS, Email)
- CTA: Redeem code → conversion funnel

### **Engagement Mechanics**

| Mechanic | Purpose | Implementation |
|----------|---------|-----------------|
| **Combo Counter** | Reward consecutive hits | Track success count, visual multiplier (2x, 3x, 5x) |
| **Progress Bar** | Show advancement | Top bar fills smoothly as scenes complete |
| **Sound Design** | Reinforce feedback | Distinct sounds for correct/wrong/achievement |
| **Confetti Bursts** | Celebrate moments | Triggered on milestone scores (100, 300, 500, 800 pts) |
| **Scene Scores** | Granular feedback | Show breakdown of points per scene at end |
| **Leaderboard Hook** | Competitive drive | Optional: "Beat high score of 950" message |
| **Unique Codes** | Viral mechanic | Different code per play = endless shares |

## CAMPAIGN STRATEGY: 3 Pillars

### **Pillar 1: Distribution (Owned + Paid)**
**Owned Channels:**
- Embed on website homepage (sticky banner)
- SMS link campaign ("Play to unlock 30% off")
- Email CTA (above-the-fold)
- QR code in stores/packaging

**Paid Channels:**
- Instagram Reels + TikTok ads (show 10-sec gameplay clip, CTA "Play")
- Google App Campaigns (drive to mobile web)
- Facebook Instant Ads (zero friction)
- WhatsApp Business (send link post-purchase)

**Partnership:**
- Food delivery apps (Uber Eats, DoorDash co-marketing)
- Gaming platforms (Poki, Miniclip integration)

---

### **Pillar 2: Virality Loops**
**Primary Loop: Social Sharing**
```
User plays → Scores 750+ → Gets unique code → Shares to 3 friends
→ Friends play → Enter shared code → 10% discount + bonus points
→ Original user gets "Refer 5, Get $20" reward
```

**Secondary Loop: Leaderboard**
```
Game end → "You ranked #47!" → "Beat your friend's 892 score"
→ Challenges friend via WhatsApp → Friend plays to win
```

**Tertiary Loop: Seasonal Events**
```
"Limited edition Halloween Pizza game" → Rare cosmetics (pumpkin toppings)
→ "Only 48 hours!" → Creates FOMO → Repeat plays
```

---

### **Pillar 3: Conversion Architecture**
**Funnel:**
1. **Hook** (Game intro): "Build the perfect pizza in 2 minutes"
2. **Engagement** (Scenes 2-8): Increasing difficulty, clear progress
3. **Reward** (Scene 9): Emotional peak ("Customer gives 5-star review!")
4. **Monetization** (Scene 10): 
   - Primary: "Unlock 20% off with code: DOM-5847"
   - Secondary: "Order now with DoorDash"
   - Tertiary: "Share for +5% bonus discount"
5. **Capture** (CTA): Click → Pre-filled discount code in checkout

**Expected Funnel Metrics:**
- Play rate (% who click link): 8-12%
- Completion rate (finish game): 45-60%
- Share rate (social): 25-35% of completers
- Redemption rate (use code): 18-25%
- AOV lift: 15-20% vs control
